Programming

Programming is core of computer science. Computer is a univeral machine that can be programmed to do essentially any computation. For a program to be useful we need to have a sequence of coherent steps and the power of the computer is to execute those steps quickly. The language that the course teaches is Python.

The reason why programming languages like Python exist is because in natural languages like English or Mandarin there's ambiguity that exists in the way its spoken and also they're verbose, but when it comes to programming languages a complicated task only requires few lines of code and described in a sequence of steps. The programs that are written in python, are executed by python in a language the computer understands.

An important construct in a Language is expressions, an Expression is a statement which equates to a value. If an expression does not follow the grammar rules of the programming language then a syntax error is thrown. The rules of a language is defined by grammar and one such grammar notation is Backus Naur Form. A grammar describes the language in a precise and concise way.

Variables and strings

A variable is a name which refers to a value, defined below is how to define an assignment. A variable refers to a value rather than equality expression in algebra 

name = expression
Name is a sequence of letters and numbers. A variable name can also have underscores. 
speed_of_light = 299792458 #speed of light is constant
days = 49 # assigned to 49 days
days = days - 1 #now has 48 days, the value the variable refers to changed now
String: Sequence of characters within quotes. 
course = 'Introduction to programming nano degree' #string within single quotes
course = "Introduction to programming nano degree" #string within double quotes
A string within double quote encapsulates a single quote and a string within a signle quote encapsulates a double quote.
Ex: '"Ada', "'Ada"
Two strings can be concatenated using '+'. 
name = 'Vishali' + 'Modali'
sum = 2 + 2 #adding 2 numbers
Indexing strings: Strings are sequences of characters. 
name = 'Vishali'
string[<expression>] #this is a single character

print name[0] # 'V'. Charcter at the first position of the string.

string[<expression>:<expression>]
strings with a sub-sequence of characters in the main string
name = 'udacity'
print name[1:-1] # 'dacit'. starts from the Index position 1 till 1 character before the end of the string.

find(): built-in procedure in python which outputs the first position of the target string in the search string. It outputs a -1 if it cannot find the target string.

Input->Function->Output

A function or procedure is a set of steps executed as one entity. The objective of having a function is to make the code less repetitive and also modular. When a function is used it is passed an input and gives out an output.
Calling a procedure: <procedure>(<input1>,<input2>..) The inputs are called arguments or opernds. A function is defined using the def keyword followed by a name and a parenthesis '()'. 

    def procedure_name (input1,input2,..):
        #steps
        
Defining a function or procedure:

def say_hello(name):
    print 'Welcome to udacity, my fellow udacian: '+name

Calling a function or procedure:
    say_hello('Vishali')
The function above is called a void function because rather than returning the greeting it is printing it. Internally python returns a None to functions or procedures which do not return a value [1]. Modified say_hello() 
def say_hello(name):
    return 'Welcome to udacity, my fellow udacian: '+name

print say_hello('Vishali')

Making decisions

Giving programs the ability to make decisions and execute code based on the output is very important. The comparision operators in python give the ability to programs to make decisions. Below are the comparision operators provided by python,

< - Less than,
> - Greater than,
== - Comparision operator,
<=- Less than equal,
>= - Greater than equal,
!= - Not equal
Python also provides a way to execute code based on test expressions. The block is executed only if the text expression is True. 

if <test expression>
    //block

Using operators to make decisions.

The function bigger takes in 2 numbers as arguments. The comparision operator '>' is used to decide which number is greater than the other. 
def bigger(number1, number2):
    if number1 > number2:
        return number1
    return number2
OR Logical operator: The statement below is executed if atleast one of the expressions is True. 
<expression1> OR <expression2>

Loops

Loops are a way to do things repetitively. The loop introduced below is while loop.

While loop.

The while loop evaluates the block of code as long as the test expression evaluates to True. The loop stops once the expression is evaluated to False. 

while <test expression1>:
        #block
Break: break is a special operator which jumps out of the loop as soon as it is executed and begins executing the code after the loop ends.

Lists

List is a sequence of anything as compared to a string which is a sequence of characters only.

Defining and accessing lists.

my_list = [] #empty list
my_list = ['u','d','a','c','i','t','y']
my_list[2:4] #displays a subset of the list.
my_list_1 = ['cat',1,'mat', ['rat','bat']]
As shown in my_list_1 is a collection of elements which are numeric, string as well as lists.

Mutating lists.

Lists support mutation, that is once the list is created the object is modified. 

p = ['h','e','l','l','o']
p[0]='y' #doing this is valid

s = 'hello' # a string s is created.
s[0] = 'y' #this operation is not allowed by python
To add values to a list, append() function can be used. 
p.append('!') #appends '!' to the end of the list
#p now has ['h', 'e', 'l', 'l', 'o', '!']
p.append(['u','d','a','c','i','t','y']) #appends the list to the list.
#p -> ['h', 'e', 'l', 'l', 'o', '!', ['u', 'd', 'a', 'c', 'i', 't', 'y']]

#concatenation
q = ['h', 'e', 'l', 'l', 'o']
r = ['y', 'e', 'l', 'l', 'o']

q = q + r #the '+' concatenates 2 lists and creates a new list.
#['h', 'e', 'l', 'l', 'o', 'y', 'e', 'l', 'l', 'o'] is the resultant list.

Aliasing.

my_list = ['u','d','a','c','i','t','y']
my_list_01 = my_list #When one variable's value is assigned to another this is called aliasing.
my_list_01.append(['r','o','c','k','s'])#Since lists are mutable any change done to my_list_01 affects my_list too.
